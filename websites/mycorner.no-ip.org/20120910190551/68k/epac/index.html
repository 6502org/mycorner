EPAC 68008. By Lee Davison.
Introduction.

The EPAC 68008 is a two thirds Eurocard sized single board computer that featured as a project in c't magazine starting in the February 1987 issue. It is similar in specification to the dart board and has the following features.

  • 68008 running at 8MHz
  • 32/64k bytes of EPROM
  • 8/32k bytes of RAM
  • Optional 4/8/16/32k bytes of RAM orEPROM
  • A 68681 DUART
  • A 68230 Parallel Interface/Timer

    • The board is very compact, measuring only 100mm x 110mm including the 64 way DIN 41612 I/O connector, and there is not much space for onboard additions.
