Introduction.

This is the Tiny BASIC that comes with the MPF-1B, which one of mine happens to be. I also got a scan of the keyboard overlay and the crib sheet but not a lot else.

Keyboard overlay.

The keyboard overlay presented here can be used to make an overlay for your keyboard, all you need to do is ..

  • Download the image, either the .zip or the .jpg.
  • Print the image so it's 150mm by 70mm, 5.9" by 2.75", in size.
  • Laminate the print or it won't last five minutes.
  • Cut round the image leaving a small, 2mm to 3mm, border.
  • Cut out the holes for the keys using the red outlines as your guide.

    • You should now have an overlay that's a fair reproduction of the original.
