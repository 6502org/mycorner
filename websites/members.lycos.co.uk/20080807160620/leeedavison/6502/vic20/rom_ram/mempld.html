This file includes the equations for the ISA slot logic as well as the ROM/RAM expansion logic. Only those lines that are needed for the ROM/RAM expansion have been rendered in bold.

By changing the lines that define the ROM and RAM any one of a number possible combinations of ROM and RAM can be accommodated.
