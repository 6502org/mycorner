HP-GL Viewer. By Lee Davison.
Introduction.

This HP-GL viewer is written in QuickBASIC 4.5 and can interpret most common HP-GL and some HP-GL2 drawing commands. QuickBASIC 4.5 was chosen purely because it's available on all the PCs I usually use. There is nothing particularly clever in the code and it should translate easily to any other language.

Features.

  • Most common line and curve commands.
  • Shapes.
  • Text plotting.
  • Autoscale for best fit to 640 x 480 screen.
  • Zoom and pan the plotted image.
  • Errrr. That's it.
    • Use.

    There are no command line options, when the program is run it will prompt for a filename, this should be the full name and extension (if any) and does not need to be any particular type.

    The file will then be processed to determine the size of the plot and any warnings or errors found will be reported, if there are any then the program will wait for a key press before proceeding. Next the file will be processed again and this time any resulting output will be drawn and will look something like the example shown.

    Once the plot is generated it can be zoomed untill it is plotted at a scale of 1:1 or it fills the screen (whichever is the bigger image) and the viewpoint can be panned left, right, up and down. The keys are ..

    KeyFunction
    HomeMoves the viewpoint fully out and centres it.
    Page upMoves the viewpoint out.
    Page downMoves the viewpoint in.
    EndMoves the viewpoint fully in.
    Cursor rightPans the viewpoint right.
    Cursor leftPans the viewpoint left.
    Cursor upPans the viewpoint up.
    Cursor downPans the viewpoint down.
    At any time use ESCAPE to quit.

    Limits are:-

  • You can't move out further than the plot image filling the screen.
  • You can't move in any closer than a scale of 1:1.
  • You can't pan beyond the edges of the plot image.
