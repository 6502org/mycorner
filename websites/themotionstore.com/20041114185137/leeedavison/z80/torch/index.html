The Torch Z80 second processor.

This is a second processor board for the BBC Micro. Apart form the board itself I have no information on this board. While there seems to be information available for the Acorn Z80 board any data on this board seems to be lacking. I would like at least to get the circuit diagram for it. So, if you have further information, or know where such can be found please e-mail me .
