Latest site news and updates. By Lee Davison.
10th December 2008.

An update to the QuickBASIC utility to read and convert Vic 20 .PRG files into text and/or html adds the option of a path for the images. Source included.
11th October 2008.

The ESR meter on the miscellania page is updated to add 400V input protection against connection to charged capacitors.
8th October 2008.

A new hardware project is written up for the miscellania page. ESR meter is a meter for measuring the equivalent series resistance of electrolytic capacitors.
5th October 2008.

Fractals, a Mandelbrot fractal generator that allows the images generated to be saved.
3rd October 2008.

Land, a simple fractal landscape generator, is updated to allow screenn size changes.
29th September 2008.

A demonstration of how to rubberband with the mouse on the EASy68K example programmes page. Also an update to PDP-8 Sim that improves function and speed.
7th September 2008.

An update to Invader 1000 on the EASy68K example programmes page changes the play slightly to better match that of the actual game.
6th September 2008.

Another new game on the EASy68K example programmes page. Invader 1000 is a recreation of the Galaxy Invader 1000 handheld game.
12th August 2008.

Yet another new program on the EASy68K example programmes page. Maze is a simple maze generator.
9th August 2008.

Another new program on the EASy68K example programmes page. Land is a simple fractal landscape generator. Images generated can be saved as bitmaps.
6th August 2008.

Another new program on the EASy68K example programmes page. PDP-8 Sim is a basic PDP-8 simulator and includes a "Hello world!" program.
30th July 2008.

A new program on the EASy68K example programmes page. Stars is a swooping, swirling starfield.

20th March 2008.

Last year was mostly 68K code so this year I'll start off with something different. A little code for the 6502 code bits page. SIN and COS are routines to calculate the 16-bit sine or cosine of an 8-bit angle.
