Introduction
This graphics card is not directly BBC micro compatible but is when called via the system ROM calls. The memory on the board is not directly accessible to the main CPU. The board contains the following...
  • EF9366P Graphics processor
  • 48 x MK5416N 16k bit DRAM
  • Misc. 74 series logic
