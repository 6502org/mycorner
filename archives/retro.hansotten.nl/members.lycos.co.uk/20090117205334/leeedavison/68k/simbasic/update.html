This page details updates which are specific to the EASy68K version of EhBASIC. For general update details go here.

27th July 2005.

New commands and functions have been added to make use of some of the enhancements to EASy68K. This includes key testing and double buffered graphics. See the language reference for details.

The version is now 3.21/2.33. Enjoy.

26th May 2004.

Nearly all the EASy68K graphics commands have been re-written. This is to give uniform handling of the parameters of all commands and reduces the code size. It also fixes an undocumented feature of the MOVE command.

The version is now 2.21/2.17. Enjoy.

25th April 2003.

Added the TI function. It returns the time as 100ths of seconds since midnight.

The version is now 2.20/2.01. Enjoy.

17th April 2003.

Added the POINT(X,Y) function.

To maintain the same version sequence as the generic version of EhBASIC68 a sub version number has been added for the EASy68k extensions. This version is therfore 1.09/1.00. Enjoy.

9th April 2003.

Added graphics commands to support the graphics routines available in EASy68k. Most of them are now directly accessible using the new BASIC commands.

The version number is now 1.06. Enjoy.
